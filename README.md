# 🏦 ATM Simulation System (Python)

This project is a simple ATM Simulation System developed using Python. It is a menu-driven application that allows users to perform basic banking operations in an interactive and user-friendly way.

## 🚀 Features
- Check account balance
- Deposit money
- Withdraw money
- View transaction history (statement)
- Continuous menu using infinite loop (`while True`)

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Clone the repository
2. Open in VS Code or any Python IDE
3. Run the program:

   ```bash
   python main.py
